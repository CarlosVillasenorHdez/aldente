# Fotos de testimonios

Sube aquí las fotos de los clientes para los testimonios de la landing page.

## Archivos esperados

| Archivo | Persona | Dimensiones recomendadas |
|---------|---------|--------------------------|
| `roberto.jpg` | Roberto Sánchez — La Fonda de Roberto | 200×200 px mínimo, cuadrada |
| `daniela.jpg` | Daniela Fuentes — La Hacienda | 200×200 px mínimo, cuadrada |
| `jorge.jpg`   | Jorge Méndez — Mariscos El Puerto | 200×200 px mínimo, cuadrada |

## Especificaciones

- Formato: JPG o WebP (JPG recomendado por compatibilidad)
- Tamaño máximo: 500 KB por imagen
- Proporción: **cuadrada obligatoria** (1:1) — se muestra en círculo de 44×44 px
- Fondo: preferiblemente neutro o natural
- Calidad: mínimo 200×200 px para que no se vea pixelado

## Cómo subir

1. Arrastra las imágenes a esta carpeta en tu computadora
2. Renómbralas exactamente como se indica en la tabla
3. Haz `git add . && git commit -m "feat: fotos de testimonios" && git push`
4. Haz Pull from Github en Rocket

## Mientras no hay fotos

Si los archivos no existen, la landing muestra automáticamente
las iniciales del nombre (RS, DF, JM) con un fondo oscuro.
No hay ningún error — simplemente usa el fallback.
